#解构和默认值的区别
##数组
```javascript
默认值
let [a,b,c,d=4] = [1,2,3]; //此处的d就有默认值
console.log(d); //4
```
##对象
```javascript
默认值

let {foo="222",bar} = {foo:"foo obj",bar:"bar obj"}
console.log(foo); // foo obj

let {foo="222",bar} = {bar:"bar obj"}
console.log(foo); // 222

别名
let {foo:f,bar} = {foo:"foo obj",bar:"bar obj"}
console.log(f); // foo obj

```
